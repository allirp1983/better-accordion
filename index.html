DEMO

After spending the better part of an afternoon looking for an accordion widget to fit my needs, I decided to just build my own. Uses jQUERY UI for a base, but it is stripped down to only the necessities to cut down on http requests. It is really simple to implement, and I left the file pretty readable so just dig in.
